Passation du Visa Vert
Avant de passer l'examen pour votre Visa Vert, voici quelques informations pertinentes :

L'examen est composé de 3 sections :
Section 1)
Elle porte principalement sur les lois.
Section 2)
Elle porte principalement sur l'aspect hors-jeu.
Section 3)
Elle porte principalement sur l'historique.

Nous vous recommandons donc de bien connaître la matière concernant les différentes sections avant de faire un essai, puisque les questions sont aléatoires.


Coût:
Le coût de l'examen est de 33 PA par essai.


Réussite de l'épreuve :
Afin de réussir l'épreuve et d'obtenir votre Visa Vert, vous devez remplir les 2 conditions suivantes :
- Vous devez obtenir un minimum de 50% dans chacune des sections.
- Vous devez obtenir une note cumulée d'au moins 60%.
